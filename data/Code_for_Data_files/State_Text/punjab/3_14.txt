Code of Criminal Procedure (Punjab Amendment) Act, 1940_Section Preamble--> State(s): Punjab An Act to amend the Code of Criminal Procedure, 1898, in its application to the Punjab.
Code of Criminal Procedure (Punjab Amendment) Act, 1940_Section 1--> State(s): Punjab This Act may be cited as the Code of Criminal Procedure (Punjab Amendment) Act, 1940.1 Received the assent of the Governor-General on 4th June, 1940, and was first published in the Punjab Government Gazette (Extraordinary), dated 19th June, 1940.
Code of Criminal Procedure (Punjab Amendment) Act, 1940_Section 2--> State(s): Punjab In sub-section (2) of section 162 of the Code of Criminal Procedure, 1898, after the figure "1872" the following words and figures shall be deemed to be added, namely:- "or to affect the provisions of section 27 of that Act".
